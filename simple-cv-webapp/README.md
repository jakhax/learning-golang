Simple CV Webapp
===================
## Description

Simple web application with Go for a CV for learning go html templates.

## Quick Start

- `git clone https://github.com/jakhax/simple-cv-webapp.git`
- `cd simple-cv-webapp`
- `touch .env && echo "PORT=8080" > .env`
- `go run main.go router.go data.go` or   `go build`
